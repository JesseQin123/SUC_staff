# 商品比价技能

## 基础信息

- 技能名称：商品比价服务
- 业务域：电商/零售
- 触发意图：商品比价、价格对比、哪个更便宜、比较两个商品价格
- 用户表达示例：
  - 帮我比较 iPhone 15 和三星S24 哪个更便宜
  - A1 和 A3 差多少钱
  - 我想看看小米14和 iPhone 15 的价格差异

## 工具要求

本技能需要使用一个根据商品名查询价格的工具。

- 工具名：`product.price_query`
- 方法：`POST`
- URL：`http://localhost:8000/api/mock/product/price-query`
- 输入：
  ```json
  {
    "product_name": "商品名称"
  }
  ```
- 输出：
  ```json
  {
    "found": true,
    "product_name": "iPhone 15",
    "display_name": "iPhone 15",
    "brand": "Apple",
    "price": 4599.0,
    "currency": "CNY",
    "spec": "128GB"
  }
  ```

## 流程说明

1. 获取用户要比较的两个商品名称。
2. 如果用户一次只提供了一个商品名称，继续追问另一个商品名称。
3. 当两个商品名称都已获得后，分别调用 `product.price_query` 查询两个商品的价格。
4. 如果某个商品未查询到，告诉用户具体哪个商品未命中，并请用户换一个名称。
5. 如果两个商品都查询成功，比较价格，说明哪个更便宜、差价是多少，并列出两个商品的价格、品牌和规格。

## 回复规则

- 不要在没有查询工具结果时编造价格。
- 两个商品都查到价格后必须给出明确比价结果。
- 如果工具不存在或无法调用，应提示需要配置商品价格查询工具，不要只回复“请稍候”。
